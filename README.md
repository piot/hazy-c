# Hazy

> uncertain or confused

Very basic Internet simulator for unreliable datagram transports (i.e UDP/IP). Useful for testing that an application will work reliably on Internet conditions.

Can simulate:

* Duplicate packets
* Out of Order packets
* Tamper packets
* Drop Packets
* Latency drift
* Latency jitter

## Upcoming features

* Throttle
